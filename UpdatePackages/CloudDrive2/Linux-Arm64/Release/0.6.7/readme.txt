V0.6.7(Web1.4.7.0)

1. 正式发布Android apk安装包，支持Android 7及以上的系统，未root的设备具有除了本地挂载外CloudDrive2所有的功能，root设备可以本地挂载，运行时会提示是否已经root，以及建议的挂载点目录，apk版本会申请获取本地存储权限、后台运行权限和通知栏权限，在root设备上会申请root权限，请运行时按提示授权，否则某些功能将无法正常使用。
2. 修复从跨网盘复制到百度网盘时，大文件会复制失败的问题
3. 修复添加本地文件夹时如果和已有网盘目录同名时显示不正常的问题
4. 修复通过web界面重命名时如果输入的内容为空会显示不正常的问题
5. 其它bug修复

V0.6.6(Web1.4.5.0)
1. 修复115上传大文件有概率无法成功反复重试的问题

V0.6.5(Web1.4.5.0)
1. 修复备份文件时实时监控模式下有可能出现文件重复上传的问题
2. Web界面回到上层目录时默认选中最后一级访问的文件夹，在文件夹较多时方便定位上次的选择
3. 修复Web版本更新后本地缓存不自动刷新，需要手动刷新的问题。该修复对今后的新Web版本有效，本次升级仍然可能需要手动刷新一次Web
4. 提升Web界面响应速度
5. 其它bug修复
