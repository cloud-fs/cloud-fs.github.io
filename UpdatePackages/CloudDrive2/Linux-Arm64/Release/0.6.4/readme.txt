V0.6.4(Web1.4.4.0)
1. 自动备份的文件过滤规则增加包含文件夹选项，该选项选中后，规则对文件夹也生效，注意：对文件夹生效意味着源文件的任意一级父目录名如果匹配规则，该规则都会生效
2. 增加备份日志，所有备份操作日志都会写入配置文件夹log目录下backup.<日期>的文件，可以自行查看，后续版本将会支持在前端显示备份日志
3. 支持非root的Android设备选择本地文件夹，可用于本地和云盘互相复制，以及实现在非root的Android设备上完整的多云盘备份功能，非root设备仍然不支持本地文件系统挂载
4. 扫描备份文件夹和批量跨盘复制文件时，优先处理最新的文件，提高处理效率
5. 修复0.6.3版本115搜索功能失效的问题
6. 修复0.6.3版本某些情况下服务可能会自动重启的问题
7. 其它功能完善和bug修复

V0.6.3(Web1.4.3.0)

1. 自动备份支持计划任务：每个备份可以添加任意多条计划任务，每个计划任务可设定小时:分钟，默认每天会按设定的时间执行一遍完整扫描，也可选择每周哪几天执行
2. 阿里云盘：修复部分用户重启电脑后阿里云盘登录失效的问题
3. 115：修复列目录前该目录被第三方客户端删除后，115会返回根目录内容，导致结果异常的问题
4. 115：修复个别文件无法下载的问题
5. WebUI从.net 8退回.net 7，解决0.6.0开始一些老版本浏览器不兼容的问题
6. 其它性能优化、界面微调、bug修复

V0.6.2(Web1.4.2.0)
1. 天翼云盘支持家庭共享盘
2. 解决天翼云盘容易掉登录的问题。
 注意：天翼云盘数据格式有变化，请已经添加过天翼云盘的用户重新扫码添加，否则看不到家庭共享盘，并且还可能掉登录
3. Google Drive支持团队盘和共享盘写操作，包括创建目录、删除、重命名、移动和上传文件
4. 自动备份：修复macOS下用Finder删除备份盘源文件夹的文件时目标文件夹没有相应动作的问题
5. 修复某些情况下本地文件上传结束后未正确关闭，导致本地文件不释放，本地文件无法删除或者修改的问题
6. WebUI：隐藏因为操作太快导致的错误信息窗口
7. WebUI：选择本地和远程文件夹窗口按文件名排序，Unix系统下方便查找
