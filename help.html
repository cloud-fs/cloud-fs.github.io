CloudDrive用户手册

安装与运行

安装完成后，用浏览器打开http://<安装CloudDrive的主机IP>:19798 打开管理界面。建议采用最新版Edge、Chrome、Safari或者Firefox浏览器，不支持IE浏览器

Windows

  1. 访问我们的官方网站下载 CloudDrive。
  2. 打开安装包，按照屏幕上的提示进行操作。
  3. 安装完成后，您可以在桌面找到CloudDrive2快捷方式，双击快捷方式或者浏览器打开http://localhost:19798。

Linux通用

  1. 如果Linux系统没有安装fuse3，请按照对应操作系统手册安装fuse3，安装好后确认fusermount3命令可以运行
  2. 如果你需要以非root用户运行CloudDrive，请修改/etc/fuse.conf，确保user_allow_other这一行前面的注释符#已经被删除掉
  3. 访问我们的官方网站下载对应架构的CloudDrive。
  4. 打开终端并导航到下载的安装包所在的目录。
  5. 运行以下命令以展开 CloudDrive：
    tar zxvf clouddrive-2-linux-$ARCH-$VERSION.tgz
  6. 进入展开后的目录，运行
    ./clouddrive
  7. 可以将clouddrive运行加入系统自启动，或者设置为系统服务，具体操作请参考操作系统帮助

Arch Linux

  1. Arch Linux 及其衍生版可以通过 AUR 包安装:
    yay -Syu clouddrive
  2. 通过 systemd 设置启动并开启自启：
    systemctl enable --now clouddrive
  3. 通过 systemd 查看运行状态：
    systemctl status clouddrive
  4. 通过默认浏览器打开：
    xdg-open http://localhost:19798
    xdg-open http://127.0.0.1:19798
    xdg-open http://[::]:19798
  5. 注意：浏览器打开慢或卡顿之类情况，请使用浏览器的无痕模式访问。
  6. 登录后挂载本地磁盘时设置 UID GID，获取当前用户 UID：
    id -u
    获取其他用户 UID：
    id -u

macOS

  1. 运行前请确保系统已安装macFUSE，并已按照提示正确设置权限
  2. 访问我们的官方网站下载对应架构的CloudDrive。
  3. 打开终端并导航到下载的安装包所在的目录。
  4. 运行以下命令以展开 CloudDrive：
    tar zxvf clouddrive-2-macos-$ARCH-$VERSION.tgz
  5. 进入展开后的目录，运行
    ./clouddrive
  6. 可以将clouddrive运行加入系统自启动，或者设置为系统服务，具体操作请参考操作系统帮助

注册

  1. 第一次运行CloudDrive
  2. 打开CloudDrive，点击右上角的“注册”按钮。
  3. 输入您的邮箱地址和密码，点击“注册”按钮。
  4. 系统将发送一封验证邮件到您的邮箱，请按照邮件中的提示进行操作。
  5. 验证成功后，您将被重定向到登录页面。

登录

  1. 打开CloudDrive，点击右上角的“登录”按钮。
  2. 输入您的邮箱地址和密码，点击“登录”按钮。
  3. 如果您的账号已经通过验证，您将被重定向到CloudDrive的主页。

1. 打开 CloudDrive，选择 "添加云存储"。

2. 在列表中选择您的云存储提供商。

3. 输入您的账号信息，然后按 "确定"。

4. 系统将验证您的账号，验证通过后，您的云存储将出现在 CloudDrive 中。

1. 在 CloudDrive 中，您可以直接操作云存储中的文件，如上传、下载、删除、移动等。

2. 如果您将云存储挂载到本地驱动器，您可以像操作本地文件一样操作云存储中的文件。

3. 请注意，删除文件将直接影响您云存储中的文件，一旦删除，可能无法恢复。

访问CloudDrive的WebDAV服务

服务器：http://<ip>:19798/dav

用户名：登录CloudDrive的用户Email，或者只填用户Email的用户名部分，不含@及以后的部分

密码：登录CloudDrive的用户密码

忘记密码

由于云端存储的用户信息是由用户本地密码加密，我们不保存用户密码，所以CloudDrive不支持修改密码，如忘记密码，可以重置密码，在登录界面点击重置按钮，按提示重置，重置将清空你在云端存储的账号信息并重置你的密码

注意：重置密码将清空你在云端存储的账号信息并重置你的密码，但不会影响你的会员计划。
